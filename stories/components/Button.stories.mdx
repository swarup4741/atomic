# Button

`'btn'` is a simple component which provides an extensible platform for creating interactive elements, which are typically clicked or tapped.

## Disabled

The disabled styling provided should suffice for any instance of the button component.

If you do override the disabled styling, do so for every button to give your user a consistent experience.

## Sizes

The amount of `padding` surrounding the content of a button and its outter edge can be controlled using three size utilities:

* `btn-sm`
* `btn-md`
* `btn-lg`

By default, buttons are sized to `btn-md`.

## Color

`btn-primary` can be added style the button using your configured primary color.

<Canvas>
  <Story
    args={{ children: 'Primary', className: 'btn btn-primary' }}
    name="Primary"
  >
    {Template.bind({})}
  </Story>
</Canvas>

A custom color can be created by defining a class within Tailwind's `utilities` layer:

```css
@layer utilities {
  .btn-red {
    @apply btn; /* btn is applied first, to overwrite its defaults */
    @apply bg-red-500 hover:bg-red-600 active:bg-red-700;
    @apply focus:ring-red-400 active:focus:ring-red-500;
    @apply text-gray-50;
  }
}
```

> The example below applies them inline, but extracting styles into a reusable class is recommended.
