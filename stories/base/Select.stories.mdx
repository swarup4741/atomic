# Selection

The [`::selection`](https://developer.mozilla.org/en-US/docs/Web/CSS/::selection) pseudo-element is configured to apply a background color to text selected by the user.

* A theme-aware background color is applied to the text selection.

> This feature can be seen by selecting text anywhere in this Storybook.
